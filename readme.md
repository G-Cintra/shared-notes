# Shared Notes

Repositório colaborativo para organizar e compartilhar materiais de estudo produzidos pelos alunos do curso.

A proposta é reunir, em um só lugar, resoluções de exercícios, resumos e notas de estudo, com possibilidade de revisão e aprimoramento coletivo ao longo do tempo.

## Objetivo

Este repositório busca:

- reunir materiais acadêmicos produzidos pelos alunos;
- facilitar correções, sugestões e melhorias;
- manter um padrão mínimo de organização;
- formar um acervo útil para estudo e consulta.

O foco é complementar aulas, bibliografia e listas, e não substituí-las.

## Escopo inicial

Neste momento, o repositório está voltado principalmente para:

- gabaritos e resoluções de exercícios de Métodos;
- resumo da matéria de macro.

Novos temas e disciplinas poderão ser incorporados no futuro.

## Como colaborar

O principal canal de colaboração é a aba **Issues** do GitHub.

As issues podem ser usadas para:

- apontar erros ou imprecisões;
- sugerir melhorias de redação ou organização;
- propor novos materiais;
- enviar contribuições para inclusão no repositório;
- sugerir temas ainda não cobertos.

Não é necessário saber usar LaTeX ou GitHub de forma avançada para colaborar.

## Como abrir uma issue

Ao abrir uma issue, procure incluir:

- um título claro;
- a disciplina ou tema;
- o arquivo, seção ou assunto a que se refere;
- uma descrição objetiva da correção, sugestão ou contribuição;
- anexos ou referências, quando forem úteis.

Contribuições preliminares também são bem-vindas.

## Manutenção do repositório

A organização e a edição dos arquivos ficam a cargo de um número reduzido de colaboradores com mais familiaridade com GitHub e LaTeX.

De forma geral, os mantenedores serão responsáveis por:

- revisar contribuições recebidas;
- padronizar e organizar os materiais;
- incorporar correções e sugestões;
- manter a estrutura do repositório consistente.

## Organização

A estrutura do repositório seguirá, em princípio, uma divisão por disciplina e por tipo de material, como:

- resoluções;
- resumos;
- notas;
- arquivos-fonte e PDFs, quando aplicável.

Essa organização poderá ser ajustada conforme o repositório crescer.

## Observação

Este é um repositório colaborativo em construção. Sugestões de conteúdo, correções e propostas de melhoria para a própria estrutura do projeto são bem-vindas.
